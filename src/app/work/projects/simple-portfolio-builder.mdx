## Overview

Predo is a personal attendance prediction application that helps students forecast their class attendance based on historical patterns. Users input their weekly class attendance data, and the system analyzes trends to predict future attendance likelihood.

## Key Features

- **Weekly Data Input**: Log daily class attendance for each period
- **Pattern Analysis**: Identify attendance trends across weeks
- **Visual Analytics**: Interactive graphs comparing attendance vs. absence patterns
- **Predictive Forecasting**: ML-based predictions for upcoming classes
- **Record Tracking**: Historical attendance data visualization

## Tech Stack

- **Python** - Core logic and data processing
- **Pandas** - Attendance data manipulation and analysis
- **Matplotlib** - Graph visualization for trend comparison
- **Streamlit** - Interactive UI and dashboard
- **Machine Learning** - Predictive modeling

## How It Works

1. Input weekly class attendance data
2. System processes data using Pandas
3. Matplotlib generates visual comparisons
4. ML model predicts future attendance patterns
5. Interactive Streamlit dashboard displays insights
